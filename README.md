# 🧹 KeyClean

**Нативное macOS-приложение для безопасной чистки клавиатуры MacBook.**

Блокирует клавиатуру, тачпад и мышь на системном уровне, чтобы вы могли спокойно протереть клавиатуру без случайных нажатий.

![Swift](https://img.shields.io/badge/Swift-5.9-orange) ![macOS](https://img.shields.io/badge/macOS-13%2B-blue) ![SwiftUI](https://img.shields.io/badge/SwiftUI-4-purple)

---

## ✨ Возможности

- **🔒 Полная блокировка ввода** — CGEvent tap перехватывает ВСЕ события клавиатуры, мыши и тачпада на уровне системы
- **🖥 Полноэкранная панель** — поверх всех окон, уровень screensaver
- **⌨️ Умный выход** — удержание обоих ⌘ (Command) на 3 секунды с визуальным прогрессом
- **🎨 Красивый дизайн** — тёмная тема, амбиентные градиенты, частицы, плавные анимации
- **🪶 Лёгкий** — нативный Swift, ~2 МБ, нулевые зависимости

## 📦 Установка

### Способ 1: Через скрипт (рекомендуется)

```bash
# 1. Скачай/клонируй проект
cd KeyClean

# 2. Дай права скрипту
chmod +x build.sh

# 3. Собери
./build.sh
```

Скрипт:
- Скомпилирует проект через `xcodebuild`
- Скопирует `KeyClean.app` на рабочий стол
- Предложит установить в `/Applications`

### Способ 2: Через Xcode

1. Открой `KeyClean.xcodeproj` в Xcode
2. Выбери схему **KeyClean** и таргет **My Mac**
3. Нажми **⌘+R** (Run) или **⌘+B** (Build)
4. Готовое приложение появится в `Products/`

## ⚠️ Первый запуск — разрешения

При первом запуске macOS попросит разрешение на **Accessibility** (Универсальный доступ).

**Это обязательно!** Без этого разрешения приложение не сможет перехватывать системные события.

### Как дать разрешение:

1. Откройте **System Settings** (Системные настройки)
2. Перейдите в **Privacy & Security** → **Accessibility**
3. Нажмите 🔒 чтобы разблокировать
4. Включите **KeyClean** в списке

## 🎮 Использование

1. **Запустите** KeyClean
2. Нажмите **Start Cleaning Mode**
3. Экран заблокируется — клавиатура, мышь, тачпад не работают
4. **Протрите клавиатуру** 🧽
5. **Зажмите оба ⌘ на 3 секунды** для выхода
6. Следите за кольцом прогресса — 100% = разблокировка

```
    └── ParticlesView.swift    — Декоративные плавающие частицы
```

### Как работает блокировка

```swift
// CGEvent tap — перехватывает события ДО того как они дойдут до любого приложения
CGEvent.tapCreate(
    tap: .cgSessionEventTap,     // На уровне сессии (вся система)
    place: .headInsertEventTap,  // Вставляем в начало цепочки
    options: .defaultTap,        // Можем блокировать (не только слушать)
    eventsOfInterest: eventMask, // Клавиатура + мышь + тачпад + скролл
    callback: { ... return nil } // return nil = событие заблокировано
)
```

Это **не** Electron-костыль с `preventDefault`. Это нативный macOS API, который физически блокирует события на уровне системы.
