# 🧹 KeyClean

**Нативное macOS-приложение для безопасной чистки клавиатуры MacBook.**

Блокирует клавиатуру, тачпад и мышь на системном уровне, чтобы вы могли спокойно протереть клавиатуру без случайных нажатий.

![Swift](https://img.shields.io/badge/Swift-5.9-orange) ![macOS](https://img.shields.io/badge/macOS-13%2B-blue) ![SwiftUI](https://img.shields.io/badge/SwiftUI-4-purple)

---

## 📦 Установка

### Через Xcode:

1. Открой `KeyClean.xcodeproj` в Xcode
2. Выбери схему **KeyClean** и таргет **My Mac**
3. Нажми **⌘+R** (Run) или **⌘+B** (Build)
4. Готовое приложение появится в `Products/`

---

## ⚠️ Первый запуск — разрешения

При первом запуске macOS попросит разрешение на **Accessibility** (Универсальный доступ).

---

## 🔧 Требования

- macOS 13.0+ (Ventura или новее)
- Xcode 15+ (или Xcode Command Line Tools)
- Apple Silicon или Intel Mac
